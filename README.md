# kotlin-lotto

## 기능 명세서

### 로또

- [ ] 구입 금액 입력 문구 출력
- [ ] 구입 금액 입력
    - [x] 구입 금액 유효성 검증
- [ ] 구입 금액만큼 로또 랜덤 생성
    - [x] 로또 한 장에 대한 번호 생성
- [ ] 로또 번호 유효성 검증
    - [ ] 로또 번호는 6개
    - [ ] 로또 번호 중복 불가
- [ ] 구입한 로또 출력
- [ ] 지난주 당첨 번호 입력 문구 출력
- [ ] 지난주 당첨 번호 입력
- [ ] 보너스 볼 입력 문구 출력
- [ ] 보너스 볼 입력
- [ ] 로또 당첨 결과 판별
    - [ ] 당첨 번호 일치 갯수 구하기
    - [ ] 보너스 볼 일치 여부 확인
- [ ] 수익률 구하기
- [ ] 당첨 통계 출력
